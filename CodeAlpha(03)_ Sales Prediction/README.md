# 📈 Sales Prediction using Python

## 📌 Project Overview

Sales Prediction is a Machine Learning project that forecasts future sales based on factors such as advertising spend, target audience, and marketing platform. This project helps businesses understand how marketing investments influence sales performance and supports better decision-making.

## 🎯 Objective

The main goal of this project is to build a predictive model that estimates future sales using historical advertising and sales data.

## 🚀 Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Selection
* Sales Forecasting using Machine Learning
* Model Evaluation
* Business Insights and Recommendations
* Data Visualization

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
## Insights 

1. No missing values were found in the dataset.

2. TV, Radio, and Newspaper advertising budgets were used as input features.

3. Linear Regression was used to predict sales.

## 📊 Workflow and Implementation

The project starts by collecting and loading the dataset into Python using Pandas. After loading the data, preprocessing techniques are applied to handle missing values, remove duplicate records, and prepare the dataset for analysis. Exploratory Data Analysis (EDA) is then performed to understand patterns, trends, and relationships between advertising spend and sales.

Important features that influence sales are selected and used to train a regression-based machine learning model. The dataset is divided into training and testing sets to evaluate the model's performance accurately. Evaluation metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), and R² Score are used to measure prediction accuracy.

The trained model is then used to predict future sales based on advertising budgets, target segments, and marketing platforms. Visualizations are created to better understand the impact of advertising on sales outcomes and to identify useful business trends.

## 📈 Expected Outcome

This project helps predict future sales and provides insights into how different advertising strategies affect business performance. The results can support businesses in optimizing marketing budgets, improving campaign effectiveness, and making data-driven decisions.
